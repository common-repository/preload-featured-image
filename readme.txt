Preload Featured Image automatically in posts to increase the PageSpeed Score.

== Description ==

This plugin is a "must-have" for websites using themes that display the Featured Image automatically at the top in single post pages.

== Features ==

* Automatically preloads the Featured Image in posts to improve PageSpeed scores.
* Customizable options to choose the image size to preload.
* Option to enable or disable the preload functionality.

== Installation ==

1. Upload the `preload-featured-image` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to 'Settings' > 'Preload Featured Image' to configure the plugin settings.

== Frequently Asked Questions ==

= Can I customize the image size to be preloaded? =
Yes, you can choose the image size to be preloaded from the plugin settings.

= How do I enable or disable the preload functionality? =
Go to 'Settings' > 'Preload Featured Image' and check/uncheck the "Enable Preload" option.
